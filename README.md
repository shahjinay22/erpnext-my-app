# ERPNext Sales & Workflow Management System

## 📌 Overview

This is a custom ERPNext application built using the Frappe Framework.
It was developed and deployed for a client to customize and manage business operations.

## 🚀 Features

* Custom DocTypes
* User & Role Management
* Workflow Automation
* Activity Logs (Audit Trail)
* Custom Scripts (JS & CSS)
* Reports & Data Management

## 🛠 Tech Stack

* ERPNext
* Frappe Framework
* Python
* MariaDB
* JavaScript

## 🌐 Live Demo

Demo URL: sesprojects.m.frappe.cloud

🔒 Login access is restricted due to client confidentiality.
Note: Hosted on a trial environment, so availability may be temporary.
